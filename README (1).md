## ✨ Features

- ✅ Add tasks quickly using the input field or by pressing **Enter**
- ☑️ Mark tasks as complete with a single click
- 🗑️ Delete individual tasks
- 🔍 Filter tasks — **All**, **Active**, **Completed**
- 📊 Live stats — Total, Done, and Remaining task count
- 🧹 Clear all completed tasks at once
- 💾 **LocalStorage** support — tasks are saved even after page refresh
- 📱 Fully **responsive** — works on mobile and desktop
